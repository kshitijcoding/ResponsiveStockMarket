Requirements
--------------
   Node 6+
   
Configuration
--------------
 - Install
 
   Clone the repository and run
   
   `npm install`
   
 - Star the development server
 
   `npm run dev`
   
   Navigate to http://localhost:8080/ to see the app running live
   
   
 - To run the Jest testing utilities 
 
   `npm test`
   
- To run the Production build

   `npm run build`
